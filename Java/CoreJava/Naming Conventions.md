**Note :** Always follow ==CamelCasing==

- Classes and Interfaces
	- Camel Casing
	- Ex : HelloWorld
- Variables and Methods
	- should start with lower case
	- Ex : getName()
-  Constants
	- Full uppercase
	- Ex : PIE , E

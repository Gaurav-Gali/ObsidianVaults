
**Showing all conda environments**
```shell
conda env list
```

**Activating environments**
```shell
conda activate my-env
```

**Removing environments**
```shell
conda env remove -n my-env
```

**Creating environments**
```shell
conda create --name my-env python=3.12
```

**Adding packages**
```shell
conda install numpy
```

**Removing packages**
```shell
conda remove numpy
```

1. A program to find if a number is prime or not.

**Flowchart**
![[flowchart.png]]

**Pseudocode**
```txt
1.Start
2.Read N
3.Take div = 2
4.While div < N do
	5.If div is divisible by N then
		6.Print "Not Prime"
		7.END
	8.Else
		9.div = div + 1
10.Print "Prime"
11.Exit
```
